Ground reference data catalog

The data catalog is in active development, with more data and improved functionality on the way. We welcome feedback and suggestions. Please [contact us](/about/#contact-us). The OFO ground reference data catalog consists of large forest inventory plots in which individual trees have been mapped and measured using traditional ground-based mapping methods. The datasets hosted here are generously shared by contributors under open licensing terms (details on each dataset page).
