Ground reference data catalog

The ground reference data catalog consists of large forest inventory plots in which individual trees have been mapped and measured using traditional ground-based mapping methods. They generally consist of point files (.gpkg) for use in GIS. The datasets hosted here are generously shared by contributors under open licensing terms.
